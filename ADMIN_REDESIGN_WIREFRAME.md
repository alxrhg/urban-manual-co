## Component States

### Button States
- Default: Base color, no shadow
- Hover: Darker shade, subtle shadow
- Active: Pressed state, darker
- Disabled: 50% opacity, no interaction
- Loading: Spinner, disabled state

### Card States
- Default: White/dark background, border
- Hover: Subtle shadow increase
- Selected: Accent border, background tint
- Loading: Skeleton animation

### Input States
- Default: Border, placeholder text
- Focus: Accent border, ring
- Error: Red border, error message
- Disabled: Grayed out, no interaction
