# @urban/config

Shared tooling presets for the Turborepo workspace.

## Exports
- `@urban/config/eslint` → Base ESLint flat config used by every app/package.
- `@urban/config/tailwind` → Placeholder for a shared Tailwind v4 config (to be implemented).

All configs should be authored here so we only update linting/tailwind/type settings once.
