# Lovably Design - Quick Reference

## Visual Comparison

### Lovably.com
- **Background**: Light gray (#F2F2F2)
- **Text**: Black on light gray
- **Font**: Scto Grotesk A, 13px, 400 weight
- **Layout**: Max-width 1800px, 50px horizontal padding
- **Navigation**: Minimal (logo + 1 link)
- **Hero**: Full-screen, centered, large typography
- **Content**: Text-based list, no cards
- **Spacing**: Generous white space

### Current Urban Manual
- **Background**: White/Dark mode
- **Text**: Gray-900/White
- **Font**: Inter, various sizes
- **Layout**: Max-width 1280px
- **Navigation**: Multiple links, complex
- **Hero**: Search-focused, functional
- **Content**: Card-based grid
- **Spacing**: Standard spacing

## Key Differences to Address

1. **Typography Scale**: Smaller base size (13px vs 16px)
2. **Color Palette**: Light gray background vs white
3. **Layout Width**: 1800px vs 1280px
4. **Content Style**: Text lists vs cards
5. **Navigation**: Minimal vs comprehensive
6. **Spacing**: More generous white space

## Priority Implementation Order

1. ✅ Typography (font size, weight, spacing)
2. ✅ Background color (light gray)
3. ✅ Container width (1800px)
4. ✅ Navigation simplification
5. ✅ Hero section redesign
6. ✅ Content list conversion
7. ✅ Footer simplification
