# Quick Supabase CLI Commands

## Setup (One-time)
```bash
# 1. Login
npx supabase login

# 2. Link project (replace with your project ref)
npx supabase link --project-ref avdnefdfwvpjkuanhdwk
```

## Daily Use
```bash
# Check migration status
npx supabase migration list

# Apply pending migrations
npx supabase db push

# Or use the automated workflow
./scripts/supabase-migration-workflow.sh
```

## Find Your Project Reference ID
1. Go to https://supabase.com/dashboard
2. Select your project
3. Go to Settings > General
4. Copy the "Reference ID"
