# Scto Grotesk A - Font Alternatives

## Overview
Scto Grotesk A is a commercial font with a clean, geometric, Swiss design aesthetic. This document lists free and open-source alternatives that capture a similar look and feel.

## Recommended Free Alternatives

### 1. **Inter** ⭐ (Currently in use)
- **License**: SIL Open Font License (OFL)
- **Source**: [Google Fonts](https://fonts.google.com/specimen/Inter) / [rsms.me/inter](https://rsms.me/inter/)
- **Why it's good**: 
  - Open-source, optimized for screens
  - Clean geometric skeleton similar to Scto Grotesk A
  - Excellent legibility at small sizes
  - Wide range of weights (100-900)
  - Already integrated in your project
- **Best for**: Web applications, UI design
- **Similarity**: 8/10

### 2. **Work Sans**
- **License**: SIL Open Font License (OFL)
- **Source**: [Google Fonts](https://fonts.google.com/specimen/Work+Sans)
- **Why it's good**:

## Commercial Alternatives (for reference)

These are commercial fonts but are mentioned as close matches:

### **Neue Montreal** ⭐ (Personal use free, commercial license required)
- **License**: Free for personal use, commercial license required
- **Source**: [Pangram Pangram Foundry](https://pangrampangram.com/products/neue-montreal)
- **Why it's good**:
  - Versatile Grotesque typeface
  - 14 styles (7 upright + 7 italic)
  - Clean, modern aesthetic very similar to Scto Grotesk A
  - Excellent quality and character set
  - Supports Cyrillic
- **Best for**: Professional projects (with license)
- **Similarity**: 8.5/10 (very close match!)
- **Note**: Requires commercial license for business use (~$50-100+)

### Other Commercial Options:
1. **Neue Haas Grotesk** - Modern Helvetica revival (commercial)
2. **GT America** - Swiss/American grotesque hybrid (commercial)
3. **Graphik** - Geometric neutral grotesque (commercial)
4. **Maison Neue** - Neo-grotesque with Swiss tradition (commercial)

## Implementation Guide

### Using Inter (Current Setup)
Your project already uses Inter. To optimize it for Lovably-style:

```css
font-family: 'Inter', system-ui, -apple-system, sans-serif;
font-weight: 400;
letter-spacing: 0.39px; /* Match Lovably's tracking */
```

### Adding Manrope
```html
<!-- In layout.tsx -->
<link href="https://fonts.googleapis.com/css2?family=Manrope:wght@400;500;600;700&display=swap" rel="stylesheet" />
```

```css
font-family: 'Manrope', 'Inter', system-ui, sans-serif;
```

### Adding Work Sans
```html
<!-- In layout.tsx -->
<link href="https://fonts.googleapis.com/css2?family=Work+Sans:wght@400;500;600&display=swap" rel="stylesheet" />
```

```css
font-family: 'Work Sans', 'Inter', system-ui, sans-serif;
```

## Recommendation

**Stick with Inter** for the following reasons:

1. ✅ Already integrated in your project
2. ✅ Excellent quality and web optimization
3. ✅ Very similar geometric aesthetic
4. ✅ Perfect for the minimal, clean Lovably style
5. ✅ No additional font loading needed
6. ✅ Excellent readability at small sizes (13-15px)

You can enhance Inter to match Scto Grotesk A more closely by:
- Using `font-weight: 400` (regular)
- Adding `letter-spacing: 0.39px` for hero text
- Ensuring proper line-height (22-23px)

## Testing Fonts

To test different fonts, you can temporarily update your `tailwind.config.js`:

```js
fontFamily: {
  sans: ['Manrope', 'Inter', 'system-ui', '-apple-system', 'sans-serif'],
  // or
  sans: ['Work Sans', 'Inter', 'system-ui', '-apple-system', 'sans-serif'],
}
```

Then compare the visual result with the Lovably aesthetic.
